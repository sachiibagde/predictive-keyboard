# predictive-keyboard
LSTM-based predictive keyboard model using PyTorch and NLP techniques.

# Predictive Keyboard using PyTorch
This project builds a simple predictive keyboard model that suggests the next word based on previous words, similar to smartphone keyboards.
# Technologies Used
- Python
- PyTorch
- NLTK
# What the Model Does
- Tokenizes text data
- Builds a vocabulary
- Trains an LSTM model
- Predicts top 3 next-word suggestions

pip install torch nltk
